# MCP Postgres POS

Servidor MCP con Node.js + TypeScript + PostgreSQL para inventario y ventas.

## Funcionalidades

- Listar productos
- Buscar producto
- Crear producto
- Actualizar stock
- Eliminar producto
- Registrar venta
- Registrar venta por nombre
- Ver ventas de hoy
- Ver detalle de venta
- Anular venta

## Requisitos

- Node.js
- PostgreSQL
- VS Code con soporte MCP
